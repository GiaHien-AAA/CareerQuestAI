# Career Quest AI

Career Quest AI là game nhập vai định hướng nghề nghiệp sử dụng AI.

AI có thể nhập vai:

- Boss
- Khách hàng
- QA
- Project Manager
- Đồng đội
- Mentor

Người chơi thực hiện nhiệm vụ, hội thoại với AI và nhận kết quả phân tích cuối hành trình.

---

## Công nghệ

### Frontend

- React
- TypeScript
- Vite
- Tailwind CSS

### Backend

- Node.js
- Express
- Gemini API
